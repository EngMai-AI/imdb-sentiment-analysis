# 🎬 Movie Review Sentiment Analysis

An AI-powered web app that predicts whether a movie review is Positive or Negative using Machine Learning and NLP.

---

## 🚀 Project Overview
This project uses Natural Language Processing (NLP) and Machine Learning to classify movie reviews into:
- Positive 😊
- Negative 😞

---

## 🧠 Tech Stack
- Python
- Pandas
- Scikit-learn
- NLP (Text Preprocessing)
- TF-IDF Vectorizer
- Logistic Regression
- Streamlit (UI)

---

## ✨ Features
- Sentiment prediction from text
- Confidence score
- Interactive Streamlit UI
- Probability breakdown
